Go Launcher

Minecraft Offline Launcher на Go

О проекте

Go Launcher — это лёгкий и быстрый лаунчер для Minecraft, написанный на Go. Позволяет управлять версиями игры, загрузчиками (Forge, Fabric, Quilt) и модами в одном окне.

Проект распространяется в виде одного исполняемого файла — никаких сложных установок. Встроенный authlib для офлайн-режима, поддержка скинов и профилей модов для каждой версии.

Возможности

  • Установка любых версий Minecraft
  • Forge, Fabric, Quilt — одним кликом
  • Профили модов без удаления файлов
  • Поиск модов на Modrinth и CurseForge
  • Офлайн-режим со скинами
  • Встроенный Java-рантайм
  • Автоматические обновления
  • Тёмная тема и стильный интерфейс

Технологии

Go HTML/CSS Bootstrap Icons Modrinth API CurseForge API Mojang Authlib
